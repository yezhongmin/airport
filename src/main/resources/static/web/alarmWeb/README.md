##文件说明 
 ```
 css: 演示界面样式
 dhWebSdk: js sdk文件，对接只需引入该文件夹下的文件
 raw：呼叫铃声文件
 alarmWeb.html:  观看对讲演示界面
 systemcfg.html:  服务器功能演示界面
 broadcast.html： 实时广播页面
 mediaResource.html: 音频文件列表操作解码
 timerBroadcast.html：定时广播操作页面
 addBroadcastTask.html：添加定时广播页面
 
 ```
 
##使用说明

```
	1.对接前请阅读对讲接口文档，调用可参考js目录下的js文件；
	2.使用服务器功能接口需要用服务器账号登录，账号固定为"mainalc",密码为对讲服务器登录密码；
	3.由于代码中包含iframe，请不要通过file://访问网页，必需通过http://才能访问。
	
````
